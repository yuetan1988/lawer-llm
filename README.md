# LLM律师小助手

书生·浦语 (InternLM) 开源大模型实践项目。

## 项目目标

**Vision**: 大语言模型助力全面依法治国

**Objective**: 
- 短期目标: 根据已有资料，回答法律相关问题
- 长期目标: 能够根据上传资料，结合法律条文，给出缺失项与违法项
